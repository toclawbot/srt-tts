# SRT语音合成工具 - 智能分批处理方案

## 🎯 设计理念
**放弃硬性限制，采用智能分批处理**，支持任意大小的字幕文件转换。

## 🔧 实现方案

### 核心特性
- ✅ **无文件大小限制** - 支持任意大小的SRT文件
- ✅ **智能分批处理** - 自动将大文件分成小批次处理
- ✅ **内存优化** - 避免一次性加载所有内容
- ✅ **进度实时更新** - 用户可以看到详细的处理进度
- ✅ **资源管理** - 批次间延迟，避免服务器过载

### 技术实现
```python
# 分批处理配置
BATCH_SIZE = 100  # 每批处理100段字幕
batch_count = (total_segments + BATCH_SIZE - 1) // BATCH_SIZE

# 智能分批处理
for batch_index in range(batch_count):
    start_index = batch_index * BATCH_SIZE
    end_index = min((batch_index + 1) * BATCH_SIZE, total_segments)
    batch_subs = subs[start_index:end_index]
    
    # 处理当前批次
    process_batch(batch_subs)
    
    # 批次间延迟
    time.sleep(1)
```

## 🛠️ 配置调优

### 可调整参数
```python
# 分批大小（默认100）
BATCH_SIZE = 100

# 批次间延迟（默认1秒）
BATCH_DELAY = 1

# 并发批次数（默认1）
CONCURRENT_BATCHES = 1
```

### 性能调优建议
- **高内存服务器**: 可以增大BATCH_SIZE
- **低性能服务器**: 减小BATCH_SIZE，增加BATCH_DELAY
- **多核服务器**: 可以启用CONCURRENT_BATCHES
